# ShopEase – Full Stack E-Commerce Application

ShopEase is a full-stack E-commerce platform built with **React**, **Spring Boot**, and **MySQL**.  
Developed a full stack web application with separate Admin and Customer modules. 
Implemented multiple backend features and database operations using Spring Boot REST APIs and MySQL.  
Designed a responsive React frontend connected to the backend for real-time data updates.  
Integrated Razorpay payment gateway for secure and easy online payments.
  
This repository contains both the **frontend** and **backend** in one project.

## Tech Stack ##

### Frontend
- React (Vite)
- JavaScript
- HTML, CSS
- Axios

### Backend
- Java
- Spring Boot
- Spring Web, Spring Data JPA
- MySQL

### Tools
- Eclipse
- Maven
- Postman
- Spring Tool Suite
- Git & GitHub

## Key Features ##

### User Features
- User registration and login  
- View product listings  
- Add/remove items from cart  
- Place orders  
- Track order status  

### Admin Features
- Add new products  
- Update existing products  
- Delete products  

### Additional Features
- Razorpay payment integration  
- Responsive and clean UI

## How to Run the Project ##

### Backend
-cd backend
-Runs at: http://localhost:8080

### Frontend
-cd frontend
-npm install
-npm run dev
-Runs at: http://localhost:5173
